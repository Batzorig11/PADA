Semantic tags describe what content is — better for accessibility, SEO, and the next developer (often future you).

Habits to build now

HTML do's & don'ts

Do
  • Use one <h1> per page for the main title.
  • Always give <img> a descriptive alt.
  • Close every tag and nest them properly.
  • Pick the tag that matches the meaning.
Don't
  • Wrap everything in <div> out of habit.
  • Use headings just to make text big.
  • Skip alt text on images.
  • Put block elements inside a <p>.
